# Arch-node
Arch-node é uma estrutura para facilitar a inicialização de aplicações web em node. 

# Instalação
- Renomeie o arquivo .env.examples para .env.
- Crie um banco de dados com o mesmo nome do arquivo .env.
- Rode o comando `npm install` para instalar as dependências.
- Rode o comando `npm run start` para iniciar o servidor.
- Acesse [localhost:3000/user/yourname](localhost:3000/user/yourname).
